ข้อควรรู้

  • หน้าที่เรียก endpoint นี้: https://ag-office.724insure.net/compulsory/sell-policy (กำหนดการเปิด-ปิดรหัสรถยนต์การออก พ.ร.บ.)
  • Envelope เต็ม (status/draw/recordsTotal/recordsFiltered/data) ยืนยันจากเอกสาร source-derived เดิมของทีม (AGNEW docapi: docs/API Spec/CompulsorySellPolicy/) — ไม่ใช่ ApiEnvelope ทั่วไปของ AgentLove, เป็น DataTables shape เฉพาะของ endpoint นี้ ถ้าไม่มีแถวตรงเงื่อนไข data จะเป็น string ว่าง "" ไม่ใช่ []
  • IsGovernment และ IsOverrideCompanyActive ยังไม่มีอยู่ใน row projection จริงของ endpoint นี้ ณ ตอนที่ทำเอกสารนี้ (เป็น field ที่ขอเพิ่มใหม่ตาม AGNEW-Office issue #14) — ตัวอย่างข้างล่างแสดงรูปร่างเป้าหมายหลัง CR นี้เสร็จ ไม่ใช่ response จริงปัจจุบัน
  • field IsGovernment (เมื่อเพิ่มแล้ว) จะถูกใช้เป็นต้นทางข้อมูลให้ /Policy/package/list และ /Policy/v1/package/list กรองผลลัพธ์ (ผ่าน request field IsGovernment ของสอง endpoint นั้น)

POST/Master/occupation-type/sk-register/list

ข้อควรรู้

  • ดึงรายการประเภทอาชีพสำหรับ SK Register — ใช้แสดงใน dropdown ให้ผู้สมัครเลือกอาชีพ
  • ไม่ต้องส่ง Authorization header — endpoint นี้เปิดให้เรียกได้โดยไม่ต้อง login
  • ค่าที่เลือก (Occupation_Name) จะถูกเก็บใน DataJSON.PAOccupationName ของ register/save — ไม่ได้ใช้ Occupation_Id เป็น key
  • ผลลัพธ์ถูก cache ไว้ 3 ชั่วโมง
  • ต้องส่ง body ว่าง {} เพื่อหลีกเลี่ยง HTTP 411 (Length Required)

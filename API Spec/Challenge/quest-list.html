POST/Challenge/quest/list

ข้อควรรู้

  • ใช้ดึงรายการ Quest (ด่าน/เป้าหมาย) ทั้งหมดของ Challenge หนึ่ง ๆ ตาม ChallengeID พร้อมความคืบหน้า (QuestProgress) และรางวัล (RewardGrant) ของ Agent ที่ถือ Token
  • AGNEW เรียก endpoint นี้หลัง Login เข้าหน้า /agent/main เพื่อนำข้อมูลไปแสดงใน popup ประกาศ Challenge (ดู Issue #71)
  • โครงสร้างจริงของ Data เป็น array ซ้อน array: Data[0] คือ list ของ quest object (Data[0][0], Data[0][1], ...) — ตาราง field ด้านล่างอธิบายที่ระดับ quest object 1 ตัว (Quest / QuestProgress / RewardGrant[]) เพื่อความชัดเจน
  • Token ต้องเป็น access token ของ Agent ที่ login แล้ว (ผูกกับ UserID ใน QuestProgress); ค่า MinValue/MaxValue เป็นยอดเป้าหมายของด่านนั้น, ValueSummary/ValuePercent คือความคืบหน้าปัจจุบัน

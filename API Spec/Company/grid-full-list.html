POST/Company/grid/full/list

ข้อควรรู้

  • Server-side paged/filtered/sorted grid ของ IOM_GLOBAL.dbo.Company ที่ IsDelete=0 (DataTables server-side processing style)
  • Admin only — ต้องส่ง Authorization: Bearer <access_token> (field AccessToken ใน body ไม่ถูกใช้งาน ให้ omit)
  • ⚠ Response สำเร็จใช้ envelope แบบ lowercase (status/draw/recordsTotal/recordsFiltered/data) ต่างจาก endpoint อื่นที่ใช้ {Status, Message, ErrorCode, ErrorMessage, Data, Options}
  • ⚠ data เป็น "" (empty string) เมื่อไม่พบข้อมูล ไม่ใช่ []
  • recordsTotal === recordsFiltered เสมอ ไม่มีการคำนวณ unfiltered total แยก
  • ต้องส่ง order: [] และ columns: [] อย่างน้อย (ไม่ใช่ null/omit) มิฉะนั้น error 90000999
  • length ต้อง >= 1 (0 หรือค่าลบทำให้ error 90000999 จาก divide-by-zero), server cap ที่ 100 แถวเสมอ
  • Sortable fields: ID, Name, NameEn, FriendlyName, Code, RefCompanyID, IsActive, IsDelete, CreateDate, CreateUser, UpdateDate, UpdateUser
  • Filter operators ที่รองรับ: MATCH, MATCH_VALUE, NOT_MATCH, NOT_MATCH_VALUE, LIKE, BOOLEAN, BOOLEAN_SELECT, NUMBER_*/NUM_*, DATE_EQ/LT/GT/LTE/GTE
  • เมื่อเกิด error (auth/non-admin/exception) response จะสลับไปใช้ standard envelope {Status, ErrorCode, ErrorMessage, Data, Options} — codes: 90000991, null (access denied), 90000999

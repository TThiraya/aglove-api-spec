POST/Company/active

ข้อควรรู้

  • ใช้เปิด/ปิดสถานะ IsActive ของบริษัทประกัน (Company) ทีละ 1 รายการ
  • Admin only — ต้องส่ง Authorization: Bearer <access_token> ทุก request (ไม่มี field AccessToken ใน body)
  • ID คือ Company.ID (nvarchar(50)) ต้องมีความยาว 1-255 ตัวอักษร และต้องมีอยู่จริงในระบบ มิฉะนั้นได้ ErrorCode 90903000 (Data does not exist)
  • ถ้า caller ไม่ใช่ Admin จะได้ ErrorCode เป็น null (literal JSON null) พร้อม ErrorMessage "Access denie." (typo จาก source)

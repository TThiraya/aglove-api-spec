ข้อควรรู้

  • ใช้ค้นหาแพ็กเกจ PA แรงงานต่างด้าว (PHAL) เพื่อนำ Package object ที่ได้ไปใช้ใน /OrderGEI/create
  • CompanyIDS และ PackageCodeOnlineList เป็น array รองรับค้นหาได้หลายค่าพร้อมกัน
  • ผลลัพธ์ใน Data[] เรียงตาม PackageId ที่ตรงกับ PackageCodeOnlineList ที่ส่งมา
  • CompanyIDS / PackageCodeOnlineList เป็น Mandatory — ถ้าไม่ส่ง (เช่น request ว่าง) backend จะ throw NullReferenceException (ErrorCode 5902999) ดู AGNEW Issue #57

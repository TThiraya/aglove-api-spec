ข้อควรรู้

  • โครงสร้างเหมือน /Policy/package/list ทุกประการ (request DTO, validation, upstream call เดียวกัน) — อ้างอิงจาก source code จริงเช่นเดียวกับ package/list
  • ต่างกันแค่ cache key prefix (v1/package/list/...) และ error code UNEXPECTED ใช้ C601804 แทน C601805 ของ package/list — error code อื่นเหมือนกันทุกตัว (ทั้งคู่ใช้ [APIErrorHelper("Policy","01")])
  • cache condition2 ตอนเขียน cache เป็น string ว่างเสมอ (ต่างจาก package/list ที่ใส่ UserType+ModelAgent) แต่ไม่กระทบผลลัพธ์เพราะ key หลักฝังค่าดังกล่าวไว้อยู่แล้ว
  • Response ตัวอย่างในเอกสารต้นทางย่อ PackageResult เหลือ field หลัก — โครงสร้างจริงเหมือนกับ package/list ทุก field (ดู TypeScript type PolicyPackageResult ร่วมกัน)
  • ExpireDate ต้องส่งเสมอแม้ EffectiveType=FULLYEAR (บั๊ก validation order เดียวกับ package/list) และ IsGovernment มีพฤติกรรมกรอง/cache เหมือนกันทุกประการ

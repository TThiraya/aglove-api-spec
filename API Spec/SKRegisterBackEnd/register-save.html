ข้อควรรู้

  • ใช้สำหรับบันทึกข้อมูลที่แก้ไขของ Order กลับเข้าระบบ หลังจากดึงข้อมูลด้วย `/register/summary/get`
  • ฟิลด์ที่เป็น GUID (PrefixID, ProvinceID, DistrictID, SubDistrictID ฯลฯ) ต้อง resolve จาก Master API ก่อนเสมอ ห้ามกรอกเอง
  • ระบบจะสร้าง `AddressID` ใหม่ทุกครั้งที่ save และ derive `ZipCode` จาก `SubDistrictID` อัตโนมัติ

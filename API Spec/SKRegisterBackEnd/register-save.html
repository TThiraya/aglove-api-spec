ข้อควรรู้

  • Admin-only — ต้องมี Bearer token และ role Admin
  • ถ้า IsApprove=true แล้วจะเรียกไม่ได้อีก → CC05206
  • 🔍 Dev: ต้อง echo RegisterFileID / ConsentFileID / ApplicationFileID จาก summary/get กลับทุกครั้ง (required ถ้า order มีค่าอยู่ → ว่าง → CC02919)
  • 🔍 Dev: EKYCDocumentFileID / EKYCFaceFileID ไม่ใช่ request field — read-only เปลี่ยนผ่าน register/save ไม่ได้
  • DefaultAddress / DeliveryAddress: ZipCode ถูก derive จาก SubDistrictID อัตโนมัติ, AddressID สร้างใหม่ทุก save
  • OccupationID ผ่าน validation แต่ไม่ persist — ใช้ DataJSON.PAOccupationName (ชื่ออาชีพ string) แทน

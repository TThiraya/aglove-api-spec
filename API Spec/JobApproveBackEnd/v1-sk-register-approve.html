POST/JobApproveBackEnd/v1/sk-register/approve

ข้อควรรู้

  • Approve คำสมัคร SK Co-op — set IsApprove ใน local DB และส่งข้อมูลไปยัง 724Insure (Srikrung) ในครั้งเดียว
  • IsApprove ต้องเป็น true เท่านั้น — ส่ง false จะได้ AA051101
  • ApproveTaskID ต้องได้มาจาก /JobApproveBackEnd/assign ก่อน (หลังจาก assign แล้ว IsWorking = true)
  • Partial commit: Register และ ApproveTask ถูก mark approved ก่อนส่ง 724Insure — ถ้าได้ AA051219/AA051804 local DB approved แล้ว ให้ใช้ /SKRegisterBackEnd/sk/approve/send เพื่อ retry ส่ง 724Insure
  • AA051804 (SLA notify token fail) หมายความว่า approval สำเร็จแล้วแต่ response เป็น error — ปุ่ม Approve ควร disabled

POST/Customer/grid/customer/by-owner/list

ข้อควรรู้

  • Grid API (DataTables-style) สำหรับดึงรายชื่อลูกค้าตามเจ้าของ (Owner) — ใช้ Bearer token จาก AGLove API Auth
  • Filter รองรับโครงสร้าง array ซ้อน 2 ชั้น [[ {Field, Operator, Value} ]] เพื่อจัดกลุ่มเงื่อนไข
  • columns.data ต้องตรงกับชื่อ field ฝั่ง response (เช่น ID, CustomerID, IDCard, FirstName, LastName, ProvinceName) เพื่อใช้ map คอลัมน์/เรียงลำดับ
  • CustomerType: I = นิติบุคคล (บริษัท), P = บุคคลธรรมดา
  • แต่ละแถวมีข้อมูลที่อยู่ปัจจุบัน (AddressType=CURRENT) และข้อมูลคำสั่งซื้อ/กรมธรรม์ล่าสุดของลูกค้าคนนั้นแนบมาด้วย (OrderNo, OrderType, CarLicense, PackageCode)

POST/SKRegister/v1/register/payment/confirm

ข้อควรรู้

  • ใช้ยืนยันการชำระเงิน (confirm payment) ของคำสั่งซื้อ SKRegister หลังจากลงทะเบียนครบทุกขั้นตอนแล้ว
  • สร้าง QR Code สำหรับชำระเงินผ่าน Srikrung Payment Gateway — ดู QRCodeText เพื่อเปิดหน้าชำระเงิน
  • PaymentStatus เริ่มต้นเป็น Pending จนกว่าจะมี callback ยืนยันการชำระเงินสำเร็จ

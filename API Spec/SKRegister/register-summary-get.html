POST /SKRegister/register/summary/get

POST/SKRegister/register/summary/get

ข้อควรรู้

  • ดึงข้อมูลคำขอสมัคร SKRegister แบบเต็ม (Order, Payment, DefaultAddress, DeliveryAddress, Package, ReferalAgent) สำหรับฝั่งผู้สมัครเอง
  • ใช้ Bearer token ของ session ผู้สมัคร (ได้จากขั้นตอน OTP verify ของ order นั้น) — ไม่ใช่ token ของ service account ทั่วไป และผูกกับ OrderNo เดียวนั้นเท่านั้น เรียกข้าม order อื่นจะได้ ErrorCode EA10208 (Access token is invalid)
  • ข้อมูลที่ได้ใช้ประกอบก่อนเรียก /register/save เพื่อแก้ไข
